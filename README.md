## 📊 Rossmann Store Sales Forecasting

End-to-end time series forecasting project using statistical and 
machine learning models.

## Setup
1. Clone the repo
2. Create virtual environment: `python -m venv venv`
3. Install dependencies: `pip install -r requirements.txt`

## Data
Download the dataset from 
[Kaggle](https://www.kaggle.com/c/rossmann-store-sales) 
and place files in `data/raw/`

## Project Structure
- `notebooks/01_eda.ipynb` — Exploratory Data Analysis
- `notebooks/02_preprocessing.ipynb` — Feature Engineering
- `notebooks/03_classical_models.ipynb` — SARIMA/SARIMAX (coming soon)
- `notebooks/04_advanced_models.ipynb` — Prophet/LightGBM (coming soon)
- `notebooks/05_evaluation.ipynb` — Model Comparison (coming soon)
